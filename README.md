# My-First-Project
E-COMMERCE SALES DATA ANALYSIS AND
DASHBOARD (Excel Project)

## 📌 Objective
 To analyse sales data using multiple datasets such as Customer, Product, Store, and Sales
Fact
 To clean and pre-process the data to ensure accuracy and consistency
 To integrate different datasets using common keys (Customer ID, Product ID, Store)
 To extract meaningful information using formulas like XLOOKUP and IF
 To calculate key metrics such as Total Sales
 To summarize the data using Pivot Tables
 To visualize sales performance using charts and graphs
 To identify top-performing products, cities, and customers
 To generate insights that support better business decision-making.

## 📊 Dataset
- Type: E-commerce sales data analysis and dashboard
- Format: Excel (.xlsx)
- Fields: Order Date, Category, Product, Sales, Profit, Region

## 🛠 Tools Used
- Microsoft Excel
- Pivot Tables
- Charts (Bar, Line, Pie)
- Slicers (Filters)

## 🔍 Dashboard Features
- Monthly Sales Analysis
- Category-wise Sales
- Region-wise Performance
- Top Selling Products
- Profit Analysis

## 📈 Key Insights
 Sales are not evenly distributed and are concentrated among specific customers, brands,
and regions.
 The analysis helps in identifying high-performing segments and areas with growth potential.
